Which code editor for editing files ?

You can use sublime text 3 for editing the code.
 
Download link : http://www.sublimetext.com/3

How to change logo?

Open the image folder and replace the logo.png image with your logo. For retina support change the logo@2x.png, this image should be twice the size of the non retina image.
 
 
Open the index.html file in your code editor. 
Go to line no 58 and edit the site description (alt tag ).

How to change Background image?

To change the background image go to img folder and replace bg.jpg

How to create the bg-blurred.jpg image

Open photoshop , go to filter > blur > gaussian blur. Adjust the radius pixels value as per requirement.

If you don't have photoshop you can use a online service like http://www165.lunapic.com/editor/?action=blur

How to change hero image?

To change the phone image go to img folder and replace hero.png

How to integrate your mailchimp to subscribe form?

 
 
Go to php folder, inside the php folder open subscribe.php file in your code editor.
 

Step 1

Go to line no 2, change the your-api-key to your api key

You can find the mailchimp api key here api key

  1. Click your profile name to open the Account Panel and choose Account.
  2. Click the Extras drop-down menu and choose API keys.
  3. Copy an existing API key or click the Create A Key button.
  4. Label your key to keep your keys organized.

Step 2

Go to line no 3, change the your-list-id to your list id

You can find the mailchimp list id here list id

Step 3

Go to line no 11 replace us8 with your datacentre, usually found at end of api key

 

How to change Screenshot img?

To change the Screenshot images go to img folder and replace screenshot-1.jpg to screenshot-5.jpg.

For retina images replace screenshot-1@2x.jpg to screenshot-5@2x.jpg. Retina images must be twice the normal image

How to change Background video?

To change the background video go to video folder and replace the 3 videos which are named as vb.mp4, vb.ogv and vb.webm.
 
The the videos are the same but different formats which is requied. To get a video converter visit 
 
http://www.mirovideoconverter.com/

How to change the video?

Open the index.html file in your code editor. 
 
 
Go to line no 95 and change the vimeo url last string (81209488) to your url.
 
For youtube video uncomment the youtube url on line no 99 and change the youtube url last string (xxxxxxxxxxx) to your url.

How to Edit the features?

Open the index.html file in your code editor. 
 
 
Go to line no 111 and change the features.
 
To change the feature images go to img folder and replace feature-1.png , feature-2.png , feature-3.png.

How to change countdown timer?

Open main.js in the js folder.

Go to line no 84 and change the timer. Format of time is m/d/y.

How to change Background slideshow images?

To change the images go to img folder and replace the images which are named as bg-1.jpg, bg-2.jpg, bg-3.jpg

How to get the contact form working?

 
Go to php folder, inside the php folder open contact-me.php file in your code editor.
 
 
Go to line no 3 and change the email address to your email address.
 
IMPORTANT: Use a domain email to recieve mails, it won't work with any other email address. Example, if your domain is abc.com your email address will be name@abc.com

How to Edit Privacy and terms?

Go to line no 313 and change the text for privacy modal.
 
 
Go to line no 331 and change the text for terms modal.
 

How to change Twitter feed?

 

Step 1

Open the main.js file in your code editor. 
 
Go to line no 63.
Change envatomarket to your username.

Step 2

 
 
 
First, you need a consumer key and secret keys. Get one from https://dev.twitter.com/apps
 
Edit api/config.php file and replace variables with your Consumer and oAuth Keys. 
    // Consumer Key
    define('CONSUMER_KEY', 'CONSUMER_KEY_HERE');
    define('CONSUMER_SECRET', 'CONSUMER_SECRET_HERE');

    // User Access Token
    define('ACCESS_TOKEN', 'ACCESS_TOKEN_HERE');
    define('ACCESS_SECRET', 'ACCESS_SECRET_HERE');
 
Plugin doc https://github.com/sonnyt/Tweetie
 
You need a webhost for this to work.

How to change social media?

 
 
Open the index.html file in your code editor.
 
Social media links can be changed on line no 294
 
Change the value of href { change the # to say https://twitter.com/name } to correspond to the social media name.
 
 
Add more icons if needed (optional)
 
You can add more social media links if needed.
 
More icons can be found here. http://fortawesome.github.io/Font-Awesome/icons/
 
Select the icon you need , click on one of them for example linkedin-square http://fortawesome.github.io/Font-Awesome/icon/linkedin-square/ 
 
Add another line of code after line no 297 in index.html , which should look like this , <li><a href="#"><i class="fa fa-linkedin-square"></i></a></li>
 
Check out fortawesome doc for more info http://fortawesome.github.io/Font-Awesome/examples/
