02. Customize

CSS

You can easily change colors of elements in default.css located in assets / css / color folder. For other changes you'll need to edit main.css located in css folder.

Javascript

All javascript files are stored in js folder. Main JavaScript file with initializations is main.js.

Dribbble Shots

Firstly you'll need to register an application on Dribbble. Then find plugin initialization in main.js and change Client Access Token with one you got. Also you'll need to change username of dribbble account of witch shots you'll want to display. For more information please check plugin page.

Twitter Feed

On this page you'll be able to put your username as well as other settings and you'll get a custom code. Replace code you got on previous step with what we have now in template and you'll have your twitter feed on page.
